About Minimal Clothing Co.

Minimal Clothing Co. is a lightweight single–page shopping application built for COMP 3512 – Web II. The project demonstrates how modern front-end techniques can be combined to produce smooth navigation, dynamic interfaces, and a clean user experience.

All data is loaded from a JSON file and rendered with JavaScript and DOM APIs.

Key features include:

  • SPA-style navigation using JavaScript view management
  • Filter & sort system (gender, category, size, color, price)
  • Dynamic product cards and single-product view
  • Client-side cart with quantity editing, shipping options, and totals
  • Use of the HTML <dialog> element for modal UI
